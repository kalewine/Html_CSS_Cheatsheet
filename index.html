CSS Selectors

CSS Simple Selectors
Definition: used to find or select the HTML elements in order to style them with CSS; ranking system, selector specificiy, is used when there are multiple conflicting property values that point to the same element; the highest specificity wins
Selector Type Description Selector Syntax Selector Specificity
CSS Type Selectors used to match all elements of a given type or tag name the element type is used without angle brackets least specific
CSS Class Selectors used to match elements based on the contents of their class attribute a period prepends the class attribute designation, e.g. for a class of "animal," the selector would read .animal more specific
CSS ID Selector used to match elements based on the contents of their id attribute; id attribute values should be unique in the DOM a hashtag prepends the ID attribute designation, e.g. for an ID of "chihuahua," the selector would read #chihuahua very specific
!Important Rule used on declarations to orverride any other declarations for a property and ignore selector specificity !Important is used most specific
Universal Selector used to select all HTML elements on the page an asterisk if used as the universal selector, *
CSS Combinators
Definition: a CSS selector can contain more than one simple selector; a combinator can be used between the simple selectors
Combinator Name Descrition Selectors Example Explanation
Descendent Selector matches all elements that are descendants of a specified element element element div p selects all the p elements within the div sections are selected
Child Selector selects all elements that are the children of a specified element element > element div > p selects all the p elements where the parent is a div element
Adajcent Sibling Selector selects all elements that are the adjacent siblings of a specified element; must have the same parent element element + element div + p Selects all p elements that are placed immediately after div elements
General Sibling Selector selects all elements that are siblings of a specified element element1~element2 p ~ ul Selects every ul element that are preceded by a p element
Group of CSS Selectors selects multiple elements and applies the same CSS rule element1,element2 h1, h2 Selects h1 and h2 elements
